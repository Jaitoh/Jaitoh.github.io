参数| 描述
---|---
-h|帮助
--clean|清理临时文件
-D,--onedir|默认值，生成dist文件夹
-F,--onefile|dist文件夹中只生成独立的打包文件(**推荐方法**)
-i<图标文件名.ico>|指定打包程序使用的图标文件
